# FoodOrdering
外卖点餐App，主要功能包括：登录、注册、订餐（支付方式为模拟支付）、订单管理、订单评价、用户信息管理、收货地址管理、地图定位、视频监控、送餐导航、智能客服助手（图灵机器人）、二维码扫描、微信分享、App扫码下载、当前定位城市天气查询等功能。
